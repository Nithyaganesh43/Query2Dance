Most Important Concept
Using UNION
UNION merges the results of multiple SELECT statements into one combined result set. Use it to trigger dolls and lights in the same query.
07
Move dolls + turn light ON
SQL

UNION
SELECT id, name FROM light WHERE name = 'light 1';
